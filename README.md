# Iris Flower Classification – Machine Learning Project

## Project Overview
This project demonstrates how machine learning can be used to classify iris flowers into three species based on their measurements.

## Dataset
The project uses the Iris dataset which contains:
- Sepal length
- Sepal width
- Petal length
- Petal width

Target classes:
- Setosa
- Versicolor
- Virginica

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Project Workflow
1. Import Python libraries
2. Load the dataset
3. Perform Exploratory Data Analysis (EDA)
4. Split the dataset into training and testing sets
5. Train a machine learning model
6. Evaluate the model performance
7. Make predictions

## Results
The trained model successfully classifies iris flower species based on their physical measurements.
