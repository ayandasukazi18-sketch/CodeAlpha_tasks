# Iris Flower Classification 🌸

## 📌 Project Overview
This project was completed as part of my Data Science Internship at CodeAlpha.

The goal of this project is to classify Iris flowers into three species:
- Setosa
- Versicolor
- Virginica

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Machine Learning Model
- Random Forest Classifier
- Achieved high accuracy on test data

## 🚀 What I Learned
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model training and evaluation
- Confusion matrix and classification report interpretation
