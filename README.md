# Iris Flower Classification 🌸

## Project Overview
This project was completed as part of my Data Science Internship with CodeAlpha.

The goal of the project is to build a machine learning model that can classify Iris flowers into three species based on their physical measurements.

The model learns patterns from the dataset and predicts the correct species of flower using supervised machine learning techniques.

---

## Dataset
The project uses the famous Iris dataset which contains 150 samples of iris flowers.

Each sample includes the following features:

• Sepal Length  
• Sepal Width  
• Petal Length  
• Petal Width  

The target variable is the flower species:

• Setosa  
• Versicolor  
• Virginica  

The dataset is commonly used to introduce classification problems in machine learning.

## Machine Learning Concept

A classification model learns a mapping between input features and the predicted class.

Where:

X = input features (sepal length, sepal width, petal length, petal width)

f = trained machine learning model

y = predicted iris species

The model learns patterns from training data and applies them to unseen data.
